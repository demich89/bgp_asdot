# BGP ASN Converter (ASDOT ↔ ASPLAIN)

A simple, mobile-friendly, client-side BGP ASN converter tool that runs fully offline.  
Works great on GitHub Pages, local static hosting, Hugo, or any web server.

---

## Features

- Convert between ASDOT (`1.10`) and ASPLAIN (`65546`)
- History of previous conversions (stored locally in browser)
- Tap/click history items to re-run conversion
- Mobile-friendly responsive UI
- Works offline (no dependencies)
- No backend — pure HTML/JS/CSS
- GitHub Pages compatible
